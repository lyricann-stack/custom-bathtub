# Part 1｜行銷概念書

## 1. 一句話定位

**「設計師專屬的客製浴缸引擎」**——全世界獨一無二的浴缸，由 AI Agent 7×24 接洽、工廠直造，從概念到交付全流程自動化。

不打大眾市場（浴缸非衝動性消費品），以內容把**室內設計師與建築師**引流進平台，讓「設計師幫屋主選浴缸」成為我們的銷售通路。

## 2. 三條產品線的角色分工

| 產品線 | 角色 | 行銷功能 |
|---|---|---|
| ① OEM 企業網站 | 信任背書 | 證明製造實力（模具能力、出口實績），讓設計師敢下單 |
| ② 2,000+ 款浴缸展示平台 | 廣度誘餌 | 「世界最大浴缸資料庫」＝SEO 與內容素材庫，找得到就做得出 |
| ③ 全球獨一無二客製浴缸 | 高毛利轉換 | 故事性最強的旗艦線，所有內容行銷的主角 |

三條線共用同一套後端（資料庫＋Agent 工作流），行銷上互相導流：客製線製造話題 → 帶人進 2,000 款資料庫 → OEM 網站收尾建立信任。

## 3. 目標客群（TA）

**一級客群：室內設計師（Interior Designer）／建築師（Architect）**
- 他們的工作就是替屋主決定「這個房子該有什麼浴缸」——是天然的代理銷售層。
- 既有資產：手上已有 **50 位澳洲設計師名單**，作為首波 outreach 對象。

**二級客群：高端屋主**（透過設計師間接觸及）
- 例：雪梨千萬豪宅案場景——設計師代表屋主來客製，來回修改 2–3 次定案。

**暫不經營：一般大眾**
- 浴缸非衝動性消費，public 流量轉換率低；待標誌性案例出現後再藉病毒效應外擴。

## 4. 核心訊息（USP）

1. **世界只有一個**——你畫的草圖／照片，我們做成真的浴缸。
2. **2,000+ 款任你找**——找得到的款式我們都做得出來（世界最大浴缸資料庫）。
3. **工廠直造**——OEM 大廠製造實力，可開模具，價格無中間商。
4. **AI 速度**——24/7 接洽、3D 修改快速來回，從詢價到 PI 報價不過夜。

## 5. 內容策略：雙路線

Reels／短影音只有兩種東西不會被滑掉——**非常乾貨**或**非常 juicy**。兩條都做：

**路線 A｜乾貨硬核（對設計師）**
- 材質與工藝：壓克力成型、金色／特殊色工藝、模具開發過程
- 設計知識：浴缸與衛浴空間的設計趨勢、尺寸與安裝眉角
- Before/After：草圖 → 3D → 實品的完整過程紀錄
- 發佈平台：LinkedIn、YouTube、IG（設計師聚集地，Pinterest 亦可評估）

**路線 B｜Juicy 視覺（搶眼球）**
- 獨一無二造型浴缸的情境影片（入浴、奢華場景、AI model 演繹）
- 「全世界只有一個」的開箱／吊裝／安裝瞬間
- 發佈平台：IG Reels、TikTok、YouTube Shorts

**內容產製方式**：AI 產生腳本與分鏡 → AI 生成／剪輯影像 → 人工把關品牌調性。初期每週 2–3 支，重質不重量。

## 6. AI 虛擬代言人方案

公司目前網路能見度為 0——這是劣勢也是機會（可以從零設計品牌人設）。

- **做法**：用 AI 建立一個虛擬角色（設計顧問人設），固定出現在所有短影音與社群貼文中，成為品牌記憶點。
- **驗證步驟**：
  1. 角色設定（外型、名字、語氣、專業人設：浴缸設計顧問）
  2. 產 1 支測試 Reel 內部審閱（Masa 過目）
  3. 可行 → 規模化產製；不可行 → 改用真人或純產品視覺
- **風險控管**：標註 AI 生成與否依平台規範；人設不碰價格承諾，報價一律導回 Agent 流程。

## 7. 網紅／KOL 合作框架

- **目標效應**：病毒／模仿效應——「有人做了一個金色奢華的獨一無二浴缸」這類事件性內容。
- **層級規劃**：
  - 夢幻級（如 Mr. Beast 等級）：事件行銷，付費合作，預算另議——後期再評估
  - 務實級（首選）：5–10 位**家居／室內設計類**創作者（澳洲為主），合作模式＝付費置入 or 免費客製浴缸換內容
- **合作腳本**：創作者與設計師一起設計一個浴缸 → 紀錄全過程（溝通、3D 修改、生產、安裝）→ 多支短影音連載。

## 8. 首波 Outreach：50 位澳洲設計師

- **管道**：Email ＋ 社群私訊並行。
- **Impact 原則**：信件不寫公司介紹，開頭就是視覺——一張「草圖 → 成品」對比圖或 10 秒影片。
- **行動呼籲（CTA）**：「把你客戶想要的浴缸畫給我們，48 小時內回你 3D 圖。」
- **後續**：回應者直接進 AI 接洽 Agent 流程；未回應者 2 週後由 Agent 自動 follow-up 一次。

## 9. KPI（以結果為導向，不以 AI 用量為 KPI）

| 階段 | 指標 |
|---|---|
| 認知 | 社群帳號追蹤數、影片觀看／完播率 |
| 興趣 | 設計師詢價數、資料庫瀏覽深度 |
| 轉換 | 客製案進入 3D 修改階段數、PI 發出數、收訂金成交數 |
| 口碑 | 安裝完工照回傳數（$500 credit 機制）、回購／轉介設計師數 |

## 10. 行銷時程（與後端建置對齊）

| 階段 | 行銷動作 | 前提 |
|---|---|---|
| P0（6/19 前） | 概念定版（本文件）、Masa 拍板方向 | — |
| P1（2–4 週） | 開社群帳號、AI 代言人測試 Reel、Outreach 素材製作 | 基本素材到位 |
| P2（1–2 月） | 50 位設計師 outreach、KOL 洽談、每週 2–3 支影音 | 接洽 Agent 上線 |
| P3（滾動） | KOL 連載、案例內容化、外擴 public | 資料庫＋3D 引擎接得住流量 |

**節奏鐵律：行銷不能跑在後端前面。** 資料庫與 3D 修改引擎沒就緒前，不引爆大流量（避免爆紅後接不住）。

- [ ] 50 位澳洲設計師名單整理成 CRM 格式（姓名／事務所／Email／IG）

## C. 資料庫建置（核心瓶頸 — 最高優先）

- [ ] 與**阿中**約定工作時段（線上連線或飛一趟，二擇一敲定）
- [ ] 依阿中的邏輯定版 schema：欄位、分類、命名規則、3D 背景有無標記
- [ ] 規則化整理：符合邏輯者自動歸檔；例外另開資料夾逐一排除
- [ ] 先跑 50–100 款試點驗證 schema，再放量到 2,000+ 款

## D. 3D 客製引擎（獨立研究項目）

- [ ] 工具調研：能 2D→3D、可修改造型／曲面，且**有 API 可被 Agent 呼叫**（含 Farber 新模型在內列 3–5 個候選）
- [ ] 設計「阿中術語 → AI 執行」操作模式（曲線、R 角等指令化）
- [ ] 驗收標準：3–5 個真實款式端到端跑通，產出可交付的 3D 圖
- [ ] 目標：阿中單件修改 1 小時 → 3–5 分鐘

## E. Agent 工作流

- [ ] 畫出端到端 workflow 圖，標明每步的 Agent 與人工介入點：
  1. 接洽 Agent（24/7、多語言）
  2. 客製 Agent（串 3D 工具＋阿中介入點）
  3. 報價/收款 Agent（產 PI、寄信、收 50% 訂金）
  4. 生產交接（接現有工廠 OEM 流程、並櫃船務）
  5. 售後 Agent（追安裝照、發 $500 credit、照片回資料庫）
- [ ] 對帳維持人工（已確認）；照片歸檔人工 confirm 專案歸屬
